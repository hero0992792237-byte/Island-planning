# 迹划 - 作品简介

## 🎯 项目概述

**迹划** 是一款基于 AI 的智能行程规划助手，通过自然语言对话和主题推荐，为用户个性化定制旅行计划。

**项目类型**: AI Agent + 全栈应用
**技术栈**: React + TypeScript + Vite + Express + Supabase + StepFun LLM
**项目状态**: ✅ 生产就绪
**开发时间**: 2026

---

## 🌟 核心功能

### 1. AI 智能规划
- **对话式规划**: 通过自然语言对话了解用户需求（场景/人数/时间/偏好）
- **主题式规划**: 基于用户画像（兴趣/心情/预算）智能推荐4个创意主题
- **RAG 增强**: 本地知识库 + TF-IDF 检索，为 LLM 提供真实商家数据

### 2. 多渠道商家信息
- **高德 POI**: 获取商家真实地址、电话、评分、营业时间
- **美团集成**: 自动构建搜索/团购链接
- **小红书种草**: 生成探店搜索链接

### 3. 灵活节点编排
- **10种节点类型**: activity, restaurant, snack, walk, scenic, coffee, bar, shopping, photo, rest
- **智能编排**: 根据场景自动规划 3-8 个节点
- **多场景支持**: 家庭出游、朋友聚会、同事团建

### 4. 一键预订执行
- **自动预订**: 活动预订、餐厅预约、鲜花/蛋糕订购
- **实时进度**: ExecutionProgress 组件实时展示预订状态
- **部分成功**: 支持部分失败，展示预订详情

### 5. 用户系统
- **访客模式**: 无需登录即可使用（默认1000次免费额度）
- **登录权益**: 邀请好友、好友系统、云端同步
- **Supabase 集成**: 安全认证、用户管理、数据持久化

## 🛠️ 技术架构

### 前端技术栈
```
React 18 + TypeScript + Vite 5
├─ UI 组件: 自定义组件库
├─ 状态管理: React Context + useReducer
├─ 样式: Tailwind CSS 3.4
├─ 图标: Lucide React
└─ 拖拽: react-draggable
```

### 后端技术栈
```
Express 5 + TypeScript
├─ 代理服务: 美团 Skill 代理
├─ 技能集成:
│  ├─ meituan-travel (酒旅): 酒店/景点/火车票/行程规划
│  └─ meituan-venue-guide (导购): 会场授权/二维码/绑定
└─ 中间件: cors, multer
```

### AI & 数据服务
```
StepFun LLM (step-3.5-flash-2603)
├─ 对话管理: 多轮对话信息收集
├─ 行程规划: JSON 格式结构化输出
└─ 主题推荐: 个性化创意主题生成

高德地图 API
├─ POI 搜索: 地点详情查询
└─ 坐标解析: 经纬度转换

RAG (Retrieval-Augmented Generation)
├─ 分词: 中文分词 + 停用词过滤
├─ 向量化: TF-IDF 向量化
└─ 检索: 余弦相似度匹配 (topK=5)

Supabase
├─ 认证: 邮箱/密码登录
├─ 数据库: PostgreSQL (profiles, friends)
└─ 存储: 用户资料、好友关系
```

## 🏆 技术亮点

### 1. JSON 鲁棒性
- **3种提取策略**: markdown 代码块、完整 JSON 对象、截断重试
- **LLM 自我修正**: JSON 格式错误自动发回修正
- **多层容错**: 递归截断 + 类型守卫 + 空值检查

### 2. RAG 轻量实现
- **TF-IDF 向量化**: 无需训练模型，本地运行
- **余弦相似度**: 高效检索，topK=5
- **知识库注入**: 自动将真实商家数据注入 LLM prompt

### 3. 降级策略
- **高德 POI 失败**: 静默失败，仅生成链接
- **LLM JSON 失败**: 自动重试 + 自我修正
- **预订部分失败**: 停止后续步骤，展示部分成功

### 4. 用户体验
- **访客模式**: 零门槛使用
- **实时进度**: 预订状态实时更新
- **错误友好**: 清晰的错误提示和恢复建议

VITE_LLM_MODEL=step-3.5-flash-2603
