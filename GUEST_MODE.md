# 访客模式说明

## ✅ 已启用访客模式

从今天起，应用**无需登录即可直接使用**所有功能！

### 访客模式特性

- ✅ **直接使用**: 打开应用即可开始规划行程
- ✅ **免费额度**: 默认提供 1000 次免费 API 调用
- ✅ **完整功能**: 所有核心功能都可正常使用
- ✅ **可随时登录**: 点击用户中心可以登录获取更多权益

### 访客 vs 登录用户

| 功能 | 访客模式 | 登录用户 |
|------|----------|----------|
| AI 行程规划 | ✅ 1000 次 | ✅ 更多额度 |
| 保存偏好设置 | ✅ 本地保存 | ✅ 云端同步 |
| 邀请好友 | ❌ 不支持 | ✅ 可获得额外额度 |
| 好友列表 | ❌ 不支持 | ✅ 支持 |
| 个人资料 | ❌ 不支持 | ✅ 支持 |

### 如何切换到登录模式？

1. 点击右上角 **用户图标** 👤
2. 点击 **立即登录**
3. 使用 Supabase 账号登录

登录后可以享受：
- 🎁 **邀请好友**: 双方各获得 50 次免费 API 调用
- 📊 **更多额度**: 通过邀请和使用获得更多 API 调用额度
- ☁️ **云端同步**: 设置和偏好保存到云端
- 👥 **好友功能**: 和朋友一起规划行程

### 技术说明

**修改文件**:
- `src/context/AppContext.tsx` - 初始化默认访客用户
- `src/components/ExpandedView.tsx` - 移除了登录限制

**实现方式**:
```typescript
userAuth: {
  isLoggedIn: true,  // 默认已登录
  user: {
    id: 'guest-user-' + Date.now(),
    username: 'guest',
    email: 'guest@example.com',
    displayName: '访客用户',
    // ...
  },
  freeApiQuota: 1000,  // 访客额度
}
```

### 注意事项

⚠️ **访客数据说明**:
- 偏好设置保存在浏览器本地（localStorage）
- 清除浏览器数据会丢失本地设置
- 建议登录后获得完整的云端同步体验
